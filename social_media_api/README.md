# Social Media API

This is a Social Media API built with Django and Django REST Framework.

## Features

- User Registration
- User Login
- Token Authentication
- Custom User Model with additional fields:
  - Bio
  - Profile Picture
  - Followers

## Setup Instructions

### Prerequisites

- Python 3.6+
- pip
- virtualenv (optional)
